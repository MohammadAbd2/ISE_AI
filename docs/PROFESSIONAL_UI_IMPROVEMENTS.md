## ✨ What's New

### 1. 🤖 Agent Mode Toggle

**Two Modes:**

- **🤖 Agent Mode** - Autonomous coding agent that can modify files

- Context-aware placeholder text

### 2. 📊 Token Usage Display

**Location:** Bottom of Composer, below text input

**Four Token Boxes:**
1. **Input Tokens** - Tokens in your message
2. **Output Tokens** - Tokens in AI response
3. **Total Tokens** - Combined usage
4. **Available Tokens** - Remaining quota

**Visual Design:**
- Hover effects on each box
- Total box highlighted in teal
- Available box highlighted in cyan
- Monospace font for numbers
- Real-time updates as you type

## 🔧 Customization

### Change Token Limits

Edit `frontend/src/App.jsx`:
```javascript
const [tokenUsage, setTokenUsage] = useState({
  inputTokens: 0,
  outputTokens: 0,
  totalTokens: 0,
  availableTokens: 200000,  // Change this value
});
```

Edit `frontend/src/styles/global.css`:

## 🐛 Troubleshooting

### Token Display Not Showing
- Check `tokenUsage` prop is passed to Composer
- Verify state initialization in App.jsx
- Check browser console for errors

### Mode Toggle Not Working
- Verify `mode` state in App.jsx
- Check `onModeChange` handler
- Ensure Composer receives both props

### Panel Not Resizable
- Check CSS `resize: horizontal` property
- Verify `overflow: auto` is set
- Try different browser (Safari has limited resize support)

### Messages Not Full Width
- Clear browser cache
- Rebuild frontend: `npm run build`
- Check `.message-row` CSS rules
